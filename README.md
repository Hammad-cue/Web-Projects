# 🛡️ Node.js JWT Authentication Lab

A fully functional REST API demonstrating secure user authentication using **Node.js, Express, MongoDB (Mongoose), and JSON Web Tokens (JWT)**. 

This project was built to showcase modern backend practices, including async/await database connections, password hashing with bcrypt, and protected route middleware.

---

## 🚀 Technologies Used
* **Node.js & Express.js:** Server and API routing.
* **MongoDB & Mongoose:** NoSQL database and Object Data Modeling (ODM).
* **Bcrypt:** Cryptographic password hashing.
* **JSON Web Tokens (JWT):** Stateless, secure user authentication.
