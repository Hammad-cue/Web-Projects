# 🚀 Express.js Bootstrap Signup Form

A sleek, responsive signup page built with **Node.js**, **Express**, and **EJS**. This project features a modern UI using **Bootstrap 5** with floating labels and a custom gradient background.

---

## 📸 Preview

![Signup Form Screenshot](./screenshot.png)


---

## ✨ Features

* **View Engine:** EJS (Embedded JavaScript templates)
* **Styling:** Bootstrap 5 (CDN) with custom CSS overrides
* **Responsiveness:** Mobile-first design using Bootstrap Grid
* **UX:** Floating labels and hover effects on the signup card
* **Backend:** Express.js router handling GET and POST requests
